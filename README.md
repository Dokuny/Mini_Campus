# [Mini Campus](https://www.notion.so/dokuny/Mini-Campus-ea15be7f606b4ef89b79d5942abb2ea4)

> Spring Boot를 이용한 웹 기반 학습 관리 서비스

## 📜주요 기능 소개

- 어드민 백 오피스 
    - 회원 관리
    - 강의 관리
    - 수강 신청 관리
    - 배너 관리
    
- 회원
    - 로그인 / 로그아웃
    - 이메일을 이용한 비밀번호 초기화 및 회원 인증
    - 수강 목록 관리
